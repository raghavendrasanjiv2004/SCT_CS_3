# Password Strength Checker 

This project is a **Python-based password strength checker** that securely evaluates a userâ€™s password and provides feedback on its strength along with suggestions for improvement.

##  File Information
- **Code File:** `task3.py`
- **Language:** Python

##  Features
- Secure password input using `getpass`
- Checks password length
- Validates presence of:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Detects repeated characters
- Compares against common weak passwords
- Classifies passwords as:
  - Very Weak
  - Weak
  - Medium
  - Strong
- Provides suggestions to improve password security

##  Requirements
- Python 3.x  
- No external libraries required (uses built-in `re` and `getpass` modules)
